GoldScorpion Scratch Samples
============================

```
// Expressions
def a as integer

a = ( 5 + 2 )
a = ( 5 + \
    2 )

// Assignment disallowed in if expression
if a == 7 then end

// not is used instead of the bang
if not a == 7 then end

```

Base-level types:
u8, u16, u32, u64
i8, i16, i32, i64
f32, f64
bool
string
